# Are Visual

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Build documentation,

```bash
$ yarn docs:build
```

Build library via `father-build`,

```bash
$ yarn build
```
